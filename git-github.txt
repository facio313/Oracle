git / github

1. github에 회원가입하고 로그인한다.
2. git을 다운로드하고 설치한다.
3. github에서 새 repository를 만든다.
4. github/repository에 나와있는 것들을 복사한다.

(*내 git 주소
https://github.com/facio313/gitTest.git)

새로운 저장소와 폴더 간 연결 시
폴더에서 오른쪽 클릭하고 Git Bash Here 누른 다음에
새 repository 만들 때 나온 것들을 복사해서 Git Bash Here에 붙여넣는다.
echo "# gitTest" >> README.md
git init
git add README.md
git commit -m "first commit"
git branch -M main
git remote add origin https://github.com/facio313/gitTest.git
git push -u origin main

(*존재하는 저장소에 푸쉬하기
git remote add origin https://github.com/facio313/gitTest.git
git branch -M main
git push -u origin main)

5. 새로운 파일이 만들어졌을 때!
git status
git add.
git commit -m "first commit"
git push -u origin main
--> 하면 저장소에 올라감!!!
